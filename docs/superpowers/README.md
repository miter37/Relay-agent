# Design Documents

This directory contains the design process behind Relay-agent.

- `specs/` — Design specifications. Written **before** implementation.
  A spec marked `Status: Draft` is not yet approved and may change.
- `plans/` — Implementation plans derived from an approved spec.

Documents are named `YYYY-MM-DD-topic.md` and are **not** updated after
implementation — they are a historical record of intent, not current
documentation. For current behavior, see `README.md` and `manual.md`.

Contributors: please read the relevant spec before implementing a feature.
If you disagree with a design decision, open an issue rather than deviating
from an approved spec in a pull request.
